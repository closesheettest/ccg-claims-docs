🔍 Roof Permit Lookup

Upload a list of homes → pull the most recent roofing permit for each (via Shovels.ai). Your API key stays on the server; this page only takes the shared password.

Each address = 2 API calls (resolve + permits), so the free 250 covers ~125 addresses. Runs stop cleanly if the key runs out — you can still download what finished. Heads-up: the date is when roof work was permitted, not the roof's true condition, and NO PERMIT can also mean a coverage gap (Shovels covers ~85% of the US) — spot-check a few you know.
